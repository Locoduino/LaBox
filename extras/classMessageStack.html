Detailed Description

The message stack is a small location for a few messages received and ready to be send to the caller. This is a barely a list. If the string for the given index is empty, so no this index is free. Each PushMessage will push a new one in the first free index of the list. Each GetMessage will returns the first not free message content, and free the message list for this index. There is no priority in the list...
